Szükséges előzmény: JSON

A JSON5 a JSON egy kiterjesztése. Célja a könnyebb kézzel szerkeszthetőség biztosítása oly módon, hogy a könnyű feldolgozhatóság is megmaradjon.

Ennek érdekében a JSON funkciói a JavaScript lehetőségeivel egészülnek ki.

Így a JSON5 továbbra is megfelelő adattárolásra, valamint adatkommunikációra is.
