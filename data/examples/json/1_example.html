A JSON kulcs-érték párokból áll, ezeket egymástől egy vessző választja el. Minden pár egy felső szintű kapcsos zárójelpár ( { } ) közé írandó.

A kulcsok neveit macskakörmök ( " " ) közé kell írni, tőle kettősponttal (:) elválasztva következik az érték. A kulcsok neveinek egyedinek kell lennie, azaz nem lehet két egyforma kulcsnév. Az értékekre természetesen nincs ilyen megkötés.

Az értékek különböző típusúak lehetnek, ezek közül a legalapvetőbb a szöveg, amit a kulcsokhoz hasonlóan szintén idézőjelek közé írunk.

A JSON szigorú a különleges karakterekkel. Így felesleges vesszők, idézőjelek, nem szerepelhetnek a fájlban.

A JSON-ban az sortöréseknek, tördeléseknek nincsen szerepe, a struktúrát a vesszők és zárójelek adják. Emellett a kulcsok sorrendje sem számít.

Az alábbi két JSON értelmezése, megegyezik az elsővel és egymással.
